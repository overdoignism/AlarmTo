# AlarmTo

-   A simple alarm app for Windows that allows selecting the playback device   
-   Allows setting alarms for specific days of the week
-   Supports random playlists and playing your favorite music
-   Supports multiple profiles (using different folders) and can run automatically upon login

-   一個簡單的Alarm app，可以指定播放裝置，用於Windows
-   可以設置星期幾 Alarm
-   可以設置隨機播放清單，播放自己喜歡的音樂
-   可以設置多組記憶（使用不同資料夾），並在登入時自動運行
